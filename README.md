随着企业信息系统逐渐向云计算、微服务和分布式部署模式演进，服务器节点数量、运行指标类型和异常事件复杂度不断增加，传统依靠人工巡检和固定阈值告警的运维方式难以满足系统实时监控和快速诊断的需求。针对主机运行状态采集分散、历史数据不便分析、异常发现不及时以及故障诊断依赖经验等问题，本文设计并实现了一套基于 FastAPI 的轻量级分布式智能运维原型系统。

系统以后端 FastAPI 服务为核心，结合 Vue 与 ECharts 构建前端监控大屏，采用 MySQL 与 CSV 文件保存监控历史数据，并利用 psutil 完成 CPU、内存、磁盘、网络、进程和 Docker 容器等运行指标采集。在智能分析方面，系统基于 IsolationForest 与 LocalOutlierFactor 构建无监督异常检测模块，利用历史 CPU 与内存样本训练检测模型，并结合告警规则引擎完成阈值告警、告警历史记录和告警规则启停。系统同时提供 AI 诊断接口，可根据高负载进程生成故障分析和处理建议。

为展示多节点监控效果，系统设计了模拟远程 Agent，将模拟节点数据写入数据库，并在前端提供节点切换和历史趋势查询功能。测试结果表明，系统能够完成登录认证、实时指标采集、历史数据查询、异常检测、告警展示和 AI 诊断等核心功能，基本满足毕业设计对智能运维原型系统的要求。
